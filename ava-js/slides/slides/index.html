What is GraphQL - cont

  • Describe your data
  • Ask for what you want
  • Get Predictable Results

Describe your data

						
							type Project {
							  name: String
							  tagline: String
							  contributors: [User]
							}

Ask for what you want

						
							{
								project(name: "GraphQL") {
									tagline
								}
							}

Get predictable results

						
							{
								"project": {
									"tagline": "A query language for APIs"
								}
							}
