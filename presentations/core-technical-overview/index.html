Gateway

  • In a few projects, proxies the request to the api or client
  • Creates a single entry for the cluster
  • logs out requests / responses
  • Gathers metrics for prometheus/grafana (response codes / response times)

RabbitMQ

  • Queueing system
  • Used as a tool to mitigate long running tasks that may slow down response times
  • Used with MassTransit to auto handle consuming/retry/failure

Comparable/US

  • Started as a small service to handle comparables
  • Used more as a domain boundry for USA focused features
  • contains proxy for SE/MLS/PR Api's

Notification

  • Ipad Port of call
  • Runs mostly off queued messages
  • can accept http requests
